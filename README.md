# xkcdviewer

xkcdviewer is a desktop XKCD webcomic viewer for Linux.



### Dependencies :

  - PyGtk 3+


### Installation :

Clone repository: `git clone https://github.com/wa4557/xkcdreader`,
`cd ./xkcdreader`
run `python setup.py install`

That's it
